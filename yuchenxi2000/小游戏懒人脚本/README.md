# 小游戏懒人脚本

by yuchenxi2000

包含下面几个关卡：

* 锤僵尸

* 宝石迷阵

* 宝石迷阵转转看

* 隐形食脑者

* 老虎机

全程无需手操，点进关卡自动开始游戏，自动通关。PvZ还是太难了，竟然要思考，特别是那个转来转去的，脑子都要炸了，赶紧写个自动脚本给自动化了。

隐形食脑者让僵尸显形。这关我觉得最难的是开局，一开始憋不出几个豌豆，种几个豌豆发现不攻击，实在忍不住了拿寒冰菇冰一下，结果发现僵尸快进家了。

老虎机关卡只要转到四个三阳光就能通关。可以操控随机数实现，但由于操控随机数太困难了，于是直接写内存，反正也等效于操控了随机数，就这样吧。

其他小游戏好像没有特别容易自动化的，后面有想法了再更新。

## 使用方法

无其他依赖，只需按照AvZ教程安装好，运行这个脚本即可。

使用AvZ版本260224进行开发，其他版本不保证兼容性。
