# 禅境花园自动脚本

by yuchenxi2000

## 介绍

进花园自动给植物：

* 浇水

* 施肥

* 杀虫

* 放音乐

* 捡钱

此外还有这些辅助功能：

* 按Q开关脚本。主要用来防止脚本自动操作和手动操作冲突，左下角显示“AutoGarden on”说明脚本在运行，显示“AutoGarden off”说明脚本已关闭

* 按M重置商店金盏花购买时间。买了三株金盏花一段时间无法购买，此时按M就能解除限制

* 按P给一株盆栽。可以指定植物类型，在弹出窗口中输入植物编号，并勾选植物是否反向（朝左），然后设定植物颜色。此功能可以得到原版无法得到的盆栽，包括紫卡盆栽和模仿者盆栽

* 按G让所有植物立即长大。你可能要重新进一下花园来刷新植物外观

* 按I显示所有植物编号，金盏花额外显示颜色代码

* 按U关闭安全模式。此时你可以通过按P得到任意编号的盆栽，但可能导致游戏崩溃！

* 按R拯救你的花园。如果你按U得到了一颗非法植物，那么点击花园图标就会崩游戏，此时在主界面按R就能删掉所有编号不正常的植物

定义DEBUG宏进入开发者模式，此时还有以下辅助功能：

* 按P功能一样，但不会判断植物ID是否合法，注意可能导致游戏崩溃！

* 按D在命令行里输出盆栽信息

* 按T在花园里展示所有植物，共六大类植物，连续按T能在这六类里循环展示（你需要退出重进花园来刷新外观）。但不要在主玩存档使用，因为会覆盖你的所有植物！

> 植物序号可查阅本文附录
> 
> 植物颜色一般不需要填，默认0即可。如果感兴趣可以参阅本文附录
>
> 注意，玉米加农炮的盆栽判定有两格，可能会出现其他盆栽被玉米加农炮挡住的情况。被挡住的盆栽无法被选中（也就无法浇水、施肥等），这时候只能移动一下盆栽
>
> 编号33的盆栽（花盆），目前暂时无法实现自动浇水等操作，因为脚本实现细节的限制，后面可能会修复这个bug
>
> 此外，玉米加农炮盆栽有bug，用手套移动的时候不要和自身重叠，否则玉米加农炮会消失不见，只留下一个空花盆，此时只要退出花园重进就会恢复正常。bug的原因貌似是玉米加农炮占两格，但花园没做适配。此外，用推车运进蘑菇园，然后用手套也会有这个bug

使用了一些较危险的特性，如内联汇编、状态钩，如果自行改代码可能会把游戏弄崩溃，先预警一下（虽然这个脚本不会崩）。最好把该脚本作为一个独立的脚本，不要加入其他功能。

因此如果脚本失效了说明AvZ接口/内部实现改了，要不下载指定版本（260224），要不等我更新。

## 使用方法

无其他依赖，只需按照AvZ教程安装好，运行这个脚本即可。

请无视VS Code的警告，脚本能正常编译

## 附录：植物行列数

* 主花园有4行8列

* 蘑菇园只有8个位置，行数固定为0，列数按照坑位的横坐标（x）大小进行排序，分别为0~7

* 水族馆只有8个位置，行数固定为0。列数较复杂，以中间水平线分为上下两组，上面5个位置，编号0~4；下面3个，编号5~7。各组内部从左到右排序。
